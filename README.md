# Programming Core Course - Data Structures - A.Y. 2025-26
PCC2011 important experiments for data structures with their implementations for department of Artificial Intelligence and Data Science - G division.

List of experiments is as follows :
- Experiment 01 ~ Implementation of Insertion and deletion in a specific position in an Array using Function.
- Experiment 02 ~ Array Implementation of Stack.
- Experiment 03 ~ Array Implementation of Linear Queue.
- Experiment 04 ~ Array Implementation of Circular Queue. 
- Experiment 05 ~ Implement a Singly Linked List.
- Experiment 06 ~ Implementation of Stack using Linked list.
- Experiment 07 ~ Implementation of Double Ended Queue using Linked List.
- Experiment 08 ~ Implementation of Binary Search Tree and its traversal methods.
- Experiment 09 ~ Convert an Infix expression to Postfix expression using stack. 
- Experiment 10 ~ Program to Evaluate Postfix Expression using Stack ADT.
